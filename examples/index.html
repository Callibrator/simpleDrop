What is simple Drop? <-- click me to learn
simpleDrop is a simple javascript library which can be used to create drop Down menus/divs etc.
The pros for this lib is that it does not depend on any other library such as Jquery or Boostrap
Making it really easy to be used for:
  • Big project in which you do not want to import any more libraries...
  • Projects which you do not want to heavily edit the html.
  • well...
As you can see... it is very easy to create multiple drop-down menus... It is totally optional to use css... let's see some more examples.
You can use whatever HTML elements you wan't

How can I use it?
It is really simple to install and use it on your website...
You can follow the following steps
  1. Download the file
  2. Add the script to the header of your html page (eg<script src='myScripts/simpledrop-v0.0.1.js'></script>)
  3. Add your html menu code. (See below code examples)
  4. Add your js code (see below code examples)
  5. Voila. Enjoy your drop down menu-text or whatever you want
Why doesn't it work? (common mistakes)
Everyone make mistakes...
An ancient legend says that I made some of the following mistakes when I was creating the script :D
  1. Forgot to add simpleDrop.create("menuIdHere") on my script...
  2. Wrong path on html header file
  3. Forgot to add toggle-button and/or drop-toggle
  4. Forgot to set the right permissions on my files
Those mistakes can even happen to experienced programmers (Especially when it is really late... and they are tired because they are writing a library in javascript)

You need to add the script on your page header file. For example: 
<script src='lib/simpledrop-v0.0.1.js' ></script>

The following is an html file example 

<div id='codeExample'> 
	<div class='You-can-add-your-class-here toggle-button you-can-also-add-your-class-here'>
		This is the button that the user clicks to reveal or hide the content! The toggle-button class is required. you can add more classes before and/or after it.
	</div> 
	<div class='You-can-add-your-class-here drop-toggle you-can-also-add-your-class-here'>
		This is the content the the user will se when he click the button. the drop-toggle class is required for this to work... you can add more classes before and/or after it
	</div> 
</div> 
<script> 
	simpleDrop.create("codeExample")
	/*
	Be carefull here... you are using the id of the menu...
	This is the only reason that the id is required.
	*/ 
</script>
